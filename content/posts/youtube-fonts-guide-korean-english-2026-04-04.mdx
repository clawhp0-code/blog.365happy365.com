## 유튜브에서 폰트가 중요한 이유

유튜브에서 폰트는 단순한 글자가 아닙니다. **썸네일의 클릭률**, **자막의 가독성**, **채널의 브랜드 아이덴티티**를 결정하는 핵심 요소입니다. 적절한 폰트 하나가 조회수를 2~3배 올릴 수 있습니다.

이 글에서는 유튜브 플랫폼 자체의 자막 폰트부터, 영상 편집과 썸네일에 사용 가능한 한글·영문 무료 폰트까지 한눈에 볼 수 있도록 정리했습니다.

---

## 1. YouTube 플랫폼 내장 자막 폰트

YouTube 플레이어 설정에서 직접 변경할 수 있는 자막 글꼴 스타일입니다.

```
┌─────────────────────────────────────────────────────────────┐
│            YouTube 자막 폰트 패밀리 (7종)                     │
├──────────────────────┬──────────────────────────────────────┤
│  폰트 스타일           │  특징                                │
├──────────────────────┼──────────────────────────────────────┤
│  Default (기본)       │  Roboto 기반, 가장 깔끔한 기본값        │
│  Monospaced Serif    │  Courier 스타일, 코딩·타자기 느낌       │
│  Proportional Serif  │  Times 스타일, 전통적·격식 있는 느낌     │
│  Monospaced Sans     │  고정폭 고딕, 균일한 간격               │
│  Proportional Sans   │  Arial 스타일, 현대적이고 깔끔          │
│  Casual              │  손글씨 느낌, 친근하고 캐주얼            │
│  Cursive             │  필기체, 우아하고 장식적                 │
│  Small Caps          │  소문자를 작은 대문자로 표시              │
└──────────────────────┴──────────────────────────────────────┘
```

> 설정 방법: 동영상 재생 → ⚙️ 설정 → 자막/CC → 옵션 → 글꼴 스타일
