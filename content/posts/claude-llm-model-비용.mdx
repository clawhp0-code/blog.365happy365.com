model 을 변경하면서 사용하시기 바랍니다. 







지금 website 준비중인데. Openclaw 에서 Claude Code 를 사용하고 있어요.

아래와 같이 Haiku 모델에게 물어보았습니다.







🎨 웹사이트 디자인 코딩, Haiku로 충분할까요?




결론부터 말씀드리면:

"간단한 디자인 수정이나 기능 추가에는 완벽합니다! 하지만 아주 복잡한 코딩에는 Sonnet이 낫습니다."




✅ Haiku가 아주 잘하는 작업 (지금 하기에 완벽함!)




• Tailwind CSS를 이용한 색상, 폰트, 여백, 레이아웃 변경

• 버튼 위치 이동이나 크기 조절

• 기존 페이지에 새로운 텍스트나 이미지 영역 추가

• 간단한 버그 수정 및 텍스트 문구 교체

*(이런 작업들은 Haiku가 Sonnet과 거의 똑같은 퀄리티로 코드를 짜주면서 비용은 10분의 1입니다!)*




⚠️ Haiku가 어려워할 수 있는 작업 (이때는 Sonnet 추천)




• 완전히 새로운 기능(예: 복잡한 데이터베이스 연동, 결제 시스템)을 밑바닥부터 새로 짤 때

• 원인을 알 수 없는 깊고 복잡한 에러(버그)를 추적해서 고칠 때




혹시라도 Haiku 모드에서 코드를 잘 못 짜서 헤매는 모습이 보인다면, 그때 잠깐 텔레그램 채팅방 설정을 /model sonnet을 입력해서 Sonnet으로 올려서 해결하고 다시 내리시면 됩니다.
