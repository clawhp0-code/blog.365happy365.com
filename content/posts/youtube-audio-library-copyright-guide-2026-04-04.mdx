유튜브 Shorts를 만들 때 가장 신경 쓰이는 것 중 하나가 **배경음악 저작권**입니다. 잘못 사용하면 영상이 소리 없이 올라가거나, 수익이 다른 사람에게 넘어가거나, 최악의 경우 계정 제재를 받을 수 있습니다.

다행히 유튜브는 크리에이터를 위한 **공식 무료 음악 저장소**를 운영하고 있습니다 — **YouTube Audio Library**입니다.

---

## YouTube Audio Library란?

YouTube가 크리에이터에게 제공하는 **무료 배경음악·효과음 라이브러리**입니다. YouTube Studio 안에 내장되어 있으며, 수천 개의 트랙을 장르·분위기·악기별로 검색해 무료로 다운로드할 수 있습니다.

![YouTube Studio 오디오 보관함](https://upload.wikimedia.org/wikipedia/commons/thumb/e/ef/Youtube_logo.png/320px-Youtube_logo.png)

---

## 저작권 문제 없나요?

**YouTube 플랫폼 안에서 사용하는 한 저작권 문제 없습니다.** 다만 트랙마다 라이선스 조건이 다르므로 확인이 필요합니다.

```
┌───────────────────────────────────────────────────────────────────┐
│              YouTube Audio Library 라이선스 비교                   │
├────────────────────────┬──────────────────────────────────────────┤
│  라이선스 종류           │  사용 조건                               │
├────────────────────────┼──────────────────────────────────────────┤
│  저작권 없음             │  출처 표기 불필요. 완전 자유 사용           │
│  (No Attribution)      │  YouTube Shorts에 즉시 사용 가능          │
├────────────────────────┼──────────────────────────────────────────┤
│  크리에이티브 커먼즈      │  영상 설명란에 아티스트·트랙명 출처 표기    │
│  (CC-BY)               │  필요. 표기하면 자유 사용 가능             │
├────────────────────────┼──────────────────────────────────────────┤
│  YouTube 외부 플랫폼    │  트랙별로 별도 확인 필요                   │
│  (틱톡, 인스타 등)       │  일부 트랙은 외부 사용 제한될 수 있음       │
└────────────────────────┴──────────────────────────────────────────┘
```

### 주의사항

- **YouTube Shorts 수익 창출**: 일부 트랙은 특정 국가에서 수익창출이 제한될 수 있습니다. 다운로드 전 라이선스 상세 내용을 꼭 확인하세요.
- **외부 플랫폼**: 틱톡, 인스타그램 Reels 등에 같은 영상을 올릴 경우, 해당 플랫폼의 저작권 정책이 별도로 적용됩니다.
- **CC-BY 트랙**: 출처 표기를 누락하면 저작권 클레임이 들어올 수 있습니다.

## 다운로드 방법 (단계별)

### 1단계 — YouTube Studio 접속

YouTube 계정으로 로그인 후 **YouTube Studio**에 접속합니다.

```
주소: studio.youtube.com
```

> Google/YouTube 계정이 없으면 접근 불가 — 계정 생성 후 이용 가능

---

### 2단계 — 오디오 보관함 클릭

왼쪽 사이드바 메뉴에서 **오디오 보관함** 항목을 클릭합니다.

```
YouTube Studio 좌측 메뉴
├── 대시보드
├── 콘텐츠
├── 분석
├── 댓글
├── 자막
└── 오디오 보관함  ← 여기 클릭
```

---

### 3단계 — 트랙 검색 및 필터링

원하는 분위기에 맞게 필터를 설정합니다.

```
┌─────────────────────────────────────────────────────────────┐
│              오디오 보관함 필터 옵션                           │
├─────────────────┬───────────────────────────────────────────┤
│  필터 항목       │  선택 가능 옵션 (예시)                      │
├─────────────────┼───────────────────────────────────────────┤
│  장르            │  팝, 힙합, 재즈, 일렉트로닉, 클래식 등      │
│  분위기          │  밝음, 어두움, 드라마틱, 로맨틱, 행복 등     │
│  악기            │  기타, 피아노, 드럼, 신디사이저 등           │
│  재생 시간       │  60초 이하 → Shorts 최적                   │
│  저작자 표시     │  체크 해제 → 출처 표기 불필요 트랙만 표시    │
└─────────────────┴───────────────────────────────────────────┘
```

**Shorts 제작 팁**: 재생 시간 필터를 **60초 이하**로 설정하면 Shorts 길이에 딱 맞는 트랙만 나옵니다.

---

### 4단계 — 다운로드

원하는 트랙을 찾으면 트랙 오른쪽의 **다운로드 버튼 (↓)** 을 클릭합니다. `.mp3` 파일로 저장됩니다.

## 요약

1. **저작권**: YouTube 플랫폼 안에서 사용하는 한 무료·합법. CC-BY 트랙은 설명란에 출처 표기 필수.
2. **다운로드**: YouTube Studio → 오디오 보관함 → 필터 → 다운로드 버튼.
3. **Shorts 최적화**: 재생 시간 60초 이하 필터 활용.
4. **외부 플랫폼**: 틱톡·인스타 등 사용 시 해당 플랫폼 정책 별도 확인.

YouTube Audio Library를 잘 활용하면 저작권 걱정 없이 퀄리티 높은 Shorts를 꾸준히 만들 수 있습니다.
