오늘날 우리가 당연하게 여기는 **투표권**은 사실 아주 최근에 완성된 권리입니다. 불과 100년 전까지만 해도 전 세계 대부분의 나라에서 여성, 흑인, 원주민은 투표할 수 없었습니다. 이 글은 그 길고 지난한 투쟁의 역사를 정리합니다.

---

## 1부 — 여성 참정권

### 세계 최초의 물결 (1893~1930)

![1913년 워싱턴 여성 참정권 행진](https://upload.wikimedia.org/wikipedia/commons/thumb/b/b4/Suffrage_Parade_Washington_1913.jpg/500px-Suffrage_Parade_Washington_1913.jpg)

여성 참정권 운동은 두 차례의 물결로 전 세계로 퍼져나갔습니다.

| 물결 | 시기 | 주요 대상 국가 |
|:---:|:---:|:---|
| **1차 물결** | 1893 ~ 1930 | 영어권(뉴질랜드·호주·캐나다·미국), 스칸디나비아, 일부 유럽 |
| **2차 물결** | 1930 ~ 1970 | 라틴아메리카, 사하라 이남 아프리카, 프랑스·스페인·벨기에 등 유럽 후발국 |

**세계 최초 기록들**

| 기록 | 국가 | 연도 |
|:---|:---:|:---:|
| 세계 최초 여성 투표권 부여 | 🇳🇿 뉴질랜드 | **1893년** |
| 세계 최초 투표권 + 피선거권 동등 부여 | 🇫🇮 핀란드 | **1906년** |
| 미국 헌법 수정 19조 (여성 투표권) | 🇺🇸 미국 | **1920년** |
| 프랑스 여성 투표권 | 🇫🇷 프랑스 | **1944년** |

### 영국 서프러제트 운동

영국 여성들의 참정권 투쟁은 역사상 가장 극적인 사회운동 중 하나입니다.

```
┌──────────────────────────────────────────────────────────────────┐
│              영국 서프러제트 주요 전술                              │
├──────────────────┬─────────────────────────────────────────────┤
│  전술             │  내용                                        │
├──────────────────┼─────────────────────────────────────────────┤
│  거리 시위        │ 대규모 행진, 연설, 피켓 시위                  │
│  직접 행동        │ 창문 파괴, 우편함 방화, 경찰 폭행             │
│  단식 투쟁        │ 수감 중 단식 → 강제 급식으로 여론 공분 유발   │
│  자기 희생        │ 에밀리 데이비슨, 1913년 더비 경마장 달려들어 사망 |
└──────────────────┴─────────────────────────────────────────────┘
```

**단식 투쟁의 시작:** 1909년, **마리온 월리스 던롭**이 홀로웨이 교도소에서 정치범 대우를 요구하며 혼자 단식을 시작했습니다. 이후 당국의 **강제 급식**이 시작되었고, 이 잔혹한 장면이 오히려 여론을 서프러제트 편으로 돌렸습니다.

**에밀린 팽크허스트**와 딸 **크리스타벨**이 이끈 여성사회정치연합(WSPU)은 수십 차례 수감에도 굴하지 않고 투쟁을 이어갔습니다. 1918년, 영국은 마침내 30세 이상 여성에게 투표권을 부여했고, 1928년에야 남성과 동등한 21세 기준이 적용되었습니다.

### 제한적 참정권의 함정

| 국가 | 초기 제한 조건 | 완전 평등 달성 |
|:---|:---|:---:|
| 호주 | 원주민 여성 제외 | 1962년 |
| 미국 | 흑인·원주민 여성 사실상 배제 | 1965년 |
| 스위스 | 여성 참정권 자체가 늦음 | 1971년 |
| 사우디아라비아 | — | 2015년 |

> 최소 **19개국**이 처음에는 인종·나이·교육 수준·결혼 여부에 따라 일부 여성에게만 선별적으로 참정권을 부여했습니다.

## 2부 — 흑인·유색인종의 투표권 투쟁

### 헌법은 있었지만 현실은 달랐다

```
┌──────────────────────────────────────────────────────────────────┐
│              미국 헌법 수정조항과 현실의 간극                        │
├──────────┬───────────────────────────────────────────────────┤
│  연도     │  내용                                               │
├──────────┼───────────────────────────────────────────────────┤
│  1868년  │ 수정 14조 — 해방 노예에게 시민권 부여                │
│  1870년  │ 수정 15조 — 흑인 남성의 헌법적 투표권 선언           │
│  1877년  │ 재건 시대 종료 → 남부 각 주의 조직적 투표권 박탈 시작 │
│  1890년  │ 미시시피, 문해력 테스트·인두세 도입 (짐 크로 법 시작) │
│  1965년  │ 투표권법 통과 — 약 95년 만에 실질적 투표권 회복      │
└──────────┴───────────────────────────────────────────────────┘
```

### 짐 크로 법 — 투표를 막는 방법들

헌법이 투표권을 보장했음에도 남부 각 주는 다양한 방법으로 흑인 시민의 투표를 막았습니다.

| 방법 | 내용 |
|:---|:---|
| **문해력 테스트** | 복잡한 질문을 내고 틀리면 등록 거부 |
| **인두세** | 투표 전 세금 납부를 요구, 빈곤층 배제 |
| **할아버지 조항** | 1866년 이전 조상이 투표했어야만 투표 가능 |
| **백인 예비선거** | 민주당 경선을 백인만의 사적 행사로 규정 |
| **물리적 협박** | 투표 시도 시 폭력·해고·추방 위협 |

---

### 피의 일요일 — 1965년 3월 7일

앨라배마주 셀마에서 에드먼드 페터스 다리를 건너려던 평화 시위대가 주 경찰과 민병대의 곤봉·최루탄 공격을 받았습니다. 이 장면이 TV로 전국에 방영되면서 **"피의 일요일(Bloody Sunday)"**로 불리게 되었고, 미국 전역에 충격을 주었습니다.

```
1965년 3월 7일  → 피의 일요일
1965년 3월 21일 → 마틴 루터 킹 Jr. 주도 셀마-몽고메리 행진
1965년 8월 6일  → 린든 B. 존슨 대통령, 투표권법 서명
```

> 수십 년의 협박, 살인, 투쟁 끝에 흑인들의 투표소로 가는 길이 열렸습니다.

## 3부 — 원주민의 참정권 투쟁

### 시민권과 투표권은 달랐다

| 연도 | 사건 |
|:---:|:---|
| **1924년** | 인디언 시민권법 — 미국 원주민에게 시민권 부여 |
| **이후** | 각 주는 보호구역 거주, 부족 등록, 세금 납부 등을 이유로 투표 계속 거부 |
| **1940년대** | 2차 세계대전 참전 원주민 용사들이 귀국 후 투표 등록 거부당함 |
| **1948년** | 미겔 트루히요(뉴멕시코), 프랭크 해리슨(애리조나) 법정 투쟁 승리 |
| **1965년** | 투표권법으로 원주민 투표권도 실질적으로 보장 |

---

### 역설 — 영감의 원천이 배제당하다

초기 여성 참정권 운동 지도자들은 **하우데노소니(이로쿼이) 연맹 여성들**의 평등한 정치 참여에서 영감을 받았습니다. 그러나 정작 그 운동에서 **흑인·원주민 여성을 의도적으로 배제**하고 백인 중산층 여성의 권리만 우선시했습니다.

## 4부 — 한국의 경우

```
┌──────────────────────────────────────────────────────────────────┐
│              한국 참정권 역사 타임라인                               │
├──────────┬───────────────────────────────────────────────────┤
│  연도     │  내용                                               │
├──────────┼───────────────────────────────────────────────────┤
│  1910년  │ 일제 강점기 시작 — 참정권 박탈                        │
│  1945년  │ 광복                                                │
│  1948년  │ 대한민국 헌법 제정                                   │
│          │ 성별·인종 구분 없는 보통선거권 헌법에 명시             │
│          │ → 국제적으로 이른 편 (프랑스보다 4년 앞섬)            │
│  1948~   │ 6·25 전쟁, 군사 쿠데타, 권위주의 통치               │
│  1980년대│ 실질적 민주주의 실현을 위한 민주화 운동               │
│  1987년  │ 6월 항쟁 → 직선제 개헌, 실질적 참정권 회복           │
└──────────┴───────────────────────────────────────────────────┘
```

한국은 **1948년 헌법 제정과 동시에** 성별·인종 구분 없는 보통선거권을 명시했습니다. 여성 참정권 측면에서는 국제적으로 이른 편입니다. 그러나 식민 지배·전쟁·권위주의 통치를 거치면서 **실질적인 민주적 참정권 행사**는 1987년 민주화 이후에야 온전히 실현되었습니다.
