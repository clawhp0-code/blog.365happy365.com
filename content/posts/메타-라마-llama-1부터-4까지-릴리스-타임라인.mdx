메타(Meta)의 오픈소스 LLM인 라마(Llama) 시리즈에 대한 현황과 릴리스 일정도 정리해 드립니다.



구글 Gemini나 OpenAI GPT와 달리, Llama는 오픈 가중치(Open Weights) 모델이므로 구글처럼 "특정 날짜에 서버를 끄는(Shut down)" 개념의 완전한 서비스 종료가 없습니다. 다운로드만 받아두면 영구적으로 쓸 수 있지만, 생태계 지원과 클라우드 API 제공자들의 제공 여부가 자연스럽게 단종(Phase-out)을 결정합니다.



───



🦙 Meta Llama 시리즈 릴리스 타임라인 (2023~2026)



1. Llama 4 시리즈 (최신 세대 - 2025~2026)



• Llama 4 (Scout & Maverick): 2025년 4월 출시.

• 메타의 최신 플래그십 모델로 MoE(혼합 전문가) 아키텍처를 도입하고 멀티모달 기능(시각/오디오 등)을 기본 장착하여 2026년 현재 오픈소스 생태계를 주도하고 있습니다.



2. Llama 3 시리즈 (과도기 및 경량화 완성기 - 2024)



• Llama 3.3: 2024년 12월 출시. (70B 단일 모델 위주로 3.1을 크게 개선한 버전)

• Llama 3.2: 2024년 10월 출시. (비전(Vision) 추가 및 온디바이스용 경량 모델 1B/3B 출시)

• Llama 3.1: 2024년 7월 출시. (거대 405B 모델 포함, Llama 3의 완성형)

• Llama 3 (Base): 2024년 4월 출시. (8B, 70B 위주)

• 현재 상태: Llama 3.1, 3.2, 3.3은 여전히 수많은 개발자 환경과 API 제공자(Groq, Together AI, AWS 등)를 통해 널리 서비스되고 있습니다.



3. Llama 1 & Llama 2 (구세대 모델 - 2023)



• Llama 2: 2023년 7월 출시. (상업적 이용이 허용된 첫 공식 버전)

• Llama 1: 2023년 2월 출시. (연구용으로만 공개됐으나 유출로 화제)

• 현재 상태 (사실상 지원 종료): 공식적인 API 클라우드(AWS Bedrock, Azure 등)나 주요 호스팅 서비스에서는 파라미터 대비 성능 효율이 떨어지는 Llama 1과 2의 호스팅을 대부분 중단했습니다. (로컬 다운로드 및 자체 구동은 여전히 가능)

💡 Llama 생태계의 "지원 종료(Deprecation)" 특징



API 기반의 구글(Gemini)과 달리, Llama는 개발자가 모델 가중치를 소유합니다.



1. 영구 소유 가능: Meta가 "내일부로 Llama 2를 종료합니다"라고 선언하더라도, 이미 다운로드한 모델은 내 PC나 사내 서버에서 평생 구동할 수 있습니다.

2. API 제공자의 Deprecation: 직접 호스팅하지 않고 API 서비스(Groq, AWS, Azure 등)를 빌려 쓰는 경우, 해당 업체들이 구형 Llama 모델의 서비스 비용 최적화를 위해 라우팅을 중단하는 것이 실질적인 '서비스 종료'입니다.
