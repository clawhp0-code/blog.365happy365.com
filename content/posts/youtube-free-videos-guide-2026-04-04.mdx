유튜브 영상을 편집하거나 새 콘텐츠를 만들 때, 남의 영상을 무단으로 가져다 쓰면 저작권 경고(Copyright Strike)를 받을 수 있습니다. 하지만 유튜브 안에는 **처음부터 무료로 쓸 수 있도록 공개된 영상**들이 수없이 많습니다.

이 글에서는 그 영상들을 어떻게 찾고, 어떻게 올바르게 사용하는지 단계별로 설명합니다.

---

## 핵심 개념: 크리에이티브 커먼즈(CC) 라이선스란?

유튜브에서 "무료로 사용 가능한 영상"은 대부분 **크리에이티브 커먼즈(Creative Commons, CC) 라이선스**가 붙어 있습니다.

CC 라이선스는 저작권자가 "내 작품을 이런 조건 하에 자유롭게 써도 됩니다"라고 명시적으로 허락한 것입니다. 유튜브에서 제공되는 CC 라이선스는 현재 **CC BY (저작자 표시)** 하나입니다.

```
┌──────────────────────────────────────────────────────────────────┐
│              YouTube CC BY 라이선스 — 허용 범위                    │
├────────────────┬─────────────────────────────────────────────────┤
│  가능한 것      │  영상 다운로드, 편집, 재업로드, 상업적 이용         │
├────────────────┼─────────────────────────────────────────────────┤
│  반드시 해야    │  원본 제작자 출처 표기 (저작자 표시)                │
│  하는 것       │  예: "영상 출처: [채널명] (유튜브 링크)"             │
├────────────────┼─────────────────────────────────────────────────┤
│  하지 않아도   │  원작자 허락을 별도로 구할 필요 없음                  │
│  되는 것       │                                                  │
└────────────────┴─────────────────────────────────────────────────┘
```

> **주의:** CC BY는 출처 표기를 반드시 해야 합니다. 출처 표기를 생략하면 라이선스 위반입니다.

## 방법 1: YouTube 검색 필터로 CC 영상 찾기

가장 간단한 방법입니다. 유튜브 검색창에서 직접 필터를 적용하면 됩니다.

```
┌──────────────────────────────────────────────────────────────────┐
│                 CC 영상 검색 단계별 방법                            │
├─────┬────────────────────────────────────────────────────────────┤
│ 1단계│ YouTube에서 원하는 키워드 검색 (예: "nature timelapse")     │
├─────┼────────────────────────────────────────────────────────────┤
│ 2단계│ 검색 결과 페이지 상단 → [필터(Filter)] 클릭                 │
├─────┼────────────────────────────────────────────────────────────┤
│ 3단계│ 필터 메뉴에서 [기능(Features)] → [크리에이티브 커먼즈] 선택  │
├─────┼────────────────────────────────────────────────────────────┤
│ 4단계│ CC BY 라이선스가 적용된 영상만 표시됨                        │
├─────┼────────────────────────────────────────────────────────────┤
│ 5단계│ 영상 정보(설명란)에서 라이선스 조건 최종 확인                │
└─────┴────────────────────────────────────────────────────────────┘
```

![YouTube 검색 필터](https://upload.wikimedia.org/wikipedia/commons/thumb/e/ef/Youtube_logo.png/320px-Youtube_logo.png)

## 방법 2: YouTube Studio 무료 영상 클립 활용

YouTube Studio에는 크리에이터가 바로 사용할 수 있는 **공식 무료 영상 클립**이 내장되어 있습니다. 이 클립들은 출처 표기 없이도 자유롭게 사용할 수 있습니다.

```
┌──────────────────────────────────────────────────────────────────┐
│             YouTube Studio 무료 영상 클립 접근 방법                │
├─────┬────────────────────────────────────────────────────────────┤
│ 1단계│ studio.youtube.com 접속 → 로그인                          │
├─────┼────────────────────────────────────────────────────────────┤
│ 2단계│ 좌측 메뉴 → [오디오 보관함] 클릭                            │
├─────┼────────────────────────────────────────────────────────────┤
│ 3단계│ 상단 탭 [무료 효과음] 또는 영상 편집기 내 스톡 클립 확인     │
├─────┼────────────────────────────────────────────────────────────┤
│ 4단계│ 원하는 클립 선택 → 다운로드 또는 편집기에 직접 삽입          │
└─────┴────────────────────────────────────────────────────────────┘
```

> YouTube 내장 편집기(유튜브 스튜디오 → 편집 → 자르기)를 사용하면 스톡 영상 클립을 편집기 안에서 직접 불러올 수 있습니다.

## 방법 3: URL 직접 검색으로 CC 영상 모아보기

유튜브 URL에 파라미터를 추가하면 크리에이티브 커먼즈 영상만 골라서 볼 수 있습니다.

```
기본 검색 URL에 필터 파라미터 추가:

https://www.youtube.com/results?search_query=검색어&sp=EgIwAQ%3D%3D

예시:
https://www.youtube.com/results?search_query=nature+4k&sp=EgIwAQ%3D%3D
→ "nature 4k" 검색 결과 중 CC 라이선스 영상만 표시
```

## CC 영상의 올바른 출처 표기 방법

CC BY 라이선스 영상을 사용할 때는 반드시 출처를 표기해야 합니다.

```
┌──────────────────────────────────────────────────────────────────┐
│                  출처 표기 예시 (영상 설명란 기재)                   │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  영상 출처: "제목" by 채널명 (유튜브 링크)                           │
│  라이선스: Creative Commons Attribution (CC BY 4.0)              │
│                                                                  │
│  예:                                                             │
│  영상 출처: "Ocean Sunset Timelapse" by NatureVideos             │
│  (https://youtu.be/xxxxxxx)                                      │
│  License: CC BY 4.0                                              │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

```
│  사이트           │  특징                                        │
├──────────────────┼─────────────────────────────────────────────┤

```

## 주의해야 할 사항

```
┌──────────────────────────────────────────────────────────────────┐
│                   CC 영상 사용 시 주의사항                          │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ✅ 해야 하는 것                                                   │
│     - 설명란에서 라이선스 조건 직접 확인 (CC BY vs. 표준 라이선스)  │
│     - 출처 표기를 영상 설명란 또는 영상 내 크레딧에 명시            │
│     - 다운로드 전 원본 영상 설명에 "CC BY" 표기 여부 확인           │
│                                                                  │
│  ❌ 하지 말아야 하는 것                                             │
│     - 필터 없이 검색된 일반 영상을 무단으로 사용하는 것             │
│     - 출처 표기를 생략하는 것 (라이선스 위반)                       │
│     - CC BY-NC 등 다른 CC 라이선스 영상을 상업적으로 사용하는 것    │
│       (유튜브에는 CC BY만 제공되지만 외부 사이트 이용 시 주의)       │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

## 정리

유튜브에서 무료로 사용할 수 있는 영상을 찾는 방법은 크게 세 가지입니다.

1. **유튜브 검색 필터** — 검색 후 [필터] → [크리에이티브 커먼즈] 선택
2. **YouTube Studio 내장 클립** — 스튜디오 편집기 내 스톡 영상 활용
3. **Pexels·Pixabay 등 외부 CC0 사이트** — 출처 표기조차 불필요

중요한 것은 **CC BY 영상은 반드시 출처를 표기**해야 한다는 점입니다. 출처 표기만 지킨다면 상업적 사용도 가능하고, 편집·재업로드도 모두 자유롭습니다.
